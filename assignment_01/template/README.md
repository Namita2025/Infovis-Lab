# Mensa-Vis

## Setup and Start

- Clone the Git repository.
- Open the whole template folder in Visual Studio Code (VSCode, code.visualstudio.com).
- Install the extension [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) and start it by clicking on the 'Go Live' button in the bottom right corner of VSCode. It will automatically reload your website in the browser whenever a file changes (e.g., when you save); this will help you work faster and avoid testing old code that you forgot to reload.
- If it does not happen automatically, open the browser (Chrome/Edge/Firefox) at the port it tells you, by default it should be http://localhost:5500.
- Open the browser dev tools with F12.
- If not selected by default, select the console at the top of the dev tools menu bar.
